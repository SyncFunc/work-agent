# Step M7.6 安全约束 + 端到端验收 + core 不变性

## 实现方案
- **目标**：收口安全约束、跑通端到端、确认 core 不变性。
- **改动文件**：`pyproject.toml`（增 `websockets` 依赖，仅 daemon / client 路径 import）；`agent/cli.py` 子命令；`agent/daemon/*`。core 仅允许已声明的极小扩展。
- **安全**：daemon 仅绑定 `127.0.0.1`；未来可加 `hello` 携带 token 做本机鉴权（可选）。HITL 审批语义与 `TerminalTransport` 一致（`y/N`）。性能：事件转发 O(1)/事件；环形缓冲定长；`event` 消息直接 `to_dict()`，无二次序列化。
- **端到端**：起 `agent daemon`；`agent client` 连接、发任务、触发澄清 / 计划确认 / 审批经 ws 往返；`/switch` 到另一会话正常渲染。
