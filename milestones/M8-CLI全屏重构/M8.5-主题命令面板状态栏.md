# Step M8.5 主题 + 命令面板 + 状态栏

## 实现方案

- 目标：TCSS 主题（`agent/tui/tui.tcss`）、`CommandPalette`（Ctrl+P 注册 `/plan` 等）、
  `Header` 显示实时 `ctx: NN%`。一并满足旧 `调研-CLI美化方案.md` §3.1 主题诉求（更彻底）。
- 改动文件：
  - 新建 `agent/tui/tui.tcss`：定义布局与配色（暗 / 亮），对齐 Claude Code 观感。
  - `agent/tui/app.py`：`self.theme = settings.ui.theme or "textual-dark"`；
    `COMMAND_PALETTE` 注册 `/plan` / `/exec` / `/skills` / `/compact` / `/context`（绑到 `dispatch_command`）。
  - `Header`：`app.set_interval(1.0, self._refresh_ctx)` 周期读 `context_mgr.estimate_usage()`
    显示 ctx%（经 `app.call_later`），替代旧 `_status_line` 塞 prompt 前缀的 hack。
- 关键接口/算法：
  - `settings.ui.theme` 新增字段（`pydantic-settings`，见 CODEBUDDY.md §5）；映射到 Textual 主题名。
  - CommandPalette 命令 = `Command` 对象，动作 `self.dispatch_command(name)`。
- 依赖/环境：M8.2 的 `dispatch_command`；`context_mgr` 实例（现有）。
