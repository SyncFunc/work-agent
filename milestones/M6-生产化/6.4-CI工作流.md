# Step M6.4 CI 工作流（GitHub Actions + 快慢分离）

## 实现方案
- **目标**：每次 push/PR 自动跑 lint + 类型 + 测试 + 覆盖率；e2e 走 nightly/manual，不阻塞合并。
- **改动文件**：
  - 新增 `.github/workflows/ci.yml`。
  - 改 `pyproject.toml` `[project.optional-dependencies]` dev 补 `pytest-cov`、`ruff`、可选 `pytest-xdist`；`[tool.ruff]`、`[tool.basedpyright]` 配置补全；`[tool.pytest.ini_options]` 加 `cov` 相关。
- **关键流程（ci.yml）**：
  ```yaml
  on:
    push: { branches: [main] }
    pull_request: {}
    schedule: [{ cron: "0 3 * * *" }]   # nightly 跑 e2e
  jobs:
    fast:   # push/PR 门禁
      - checkout → setup-python 3.12 → pip install -e ".[dev]"
      - ruff check .
      - basedpyright
      - pytest -q                       # 默认 -m "not slow"（unit+integration）
      - pytest --cov=agent --cov-report=xml
    slow:   # nightly / workflow_dispatch
      - pytest -m slow                  # e2e 真实 LLM / fixture 重放
  ```
- **依赖/环境**：GitHub Actions 托管 runner；矩阵可加 3.13；可选 `codecov` 上传 xml。

## 验收标准
- [x] push 到 main 或开 PR 触发 `fast` job：ruff + basedpyright + pytest(unit+integration) + coverage 全绿。
- [x] `slow` job 仅在 nightly cron 或手动 `workflow_dispatch` 触发，不阻塞 PR 合并。
- [x] `pyproject.toml` dev 依赖含 `pytest-cov`/`ruff`；本地 `pytest -m slow` 可复现 CI 慢测。
- [x] 不变量：真实 LLM 调用只出现在 `slow`/e2e，绝不进 PR 门禁（防非确定阻塞合并 + 烧钱）。

## 知识沉淀
- **CI 文件**：`.github/workflows/ci.yml`。两个 job：
  - `fast`（push/PR 门禁）：`ruff check .` → `ruff format --check .` → `basedpyright` → `pytest -q`（默认 `-m 'not slow'`）→ `pytest --cov=agent --cov-report=xml -q`；矩阵 Python 3.12；`concurrency` 取消旧跑。
  - `slow`（仅 `schedule`(nightly `0 3 * * *`) 或 `workflow_dispatch`）：`pytest -m slow -q`，注入 `AGENT_LLM_API_KEY` secret；**不阻塞 PR**。
- **`pyproject.toml` 配置**：
  - `[project.optional-dependencies] dev` 补 `pytest-cov>=5.0` / `ruff>=0.5` / `pytest-xdist>=3.5`。
  - `[tool.ruff]`：`target-version="py312"`、`line-length=100`、`src=["agent"]`；`[tool.ruff.lint]` `select=[E,F,W,I,UP,B]`、`ignore=[E501]`；`[tool.ruff.lint.isort] known-first-party=["agent"]`。
  - `[tool.ruff.format]`：`quote-style="double"`。
  - `[tool.coverage.run]` `source=["agent"]` `branch=true` `omit=["tests/*","agent/testing/*"]`；`[tool.coverage.report]` `show_missing` + `exclude_lines`。
  - `[tool.basedpyright]` `typeCheckingMode="standard"`（沿用，关闭对 openai 等 SDK 的过度严格噪音）。
- **`-m 'not slow'` 约定（关键）**：`pytest.ini_options.addopts` 默认排除 e2e；PR 门禁永不跑真实 LLM。`tests/conftest.py` 与 `RecordedModel` 使 unit/integration 全确定性、零 LLM 费用。
